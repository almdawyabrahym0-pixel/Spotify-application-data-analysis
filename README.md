# 🎵 Spotify Data Analysis: Top 100 Tracks & 50 Artists

## 📌 Project Overview
هذا المشروع يقدم تحليلاً عميقاً لبيانات منصة **Spotify**، مع التركيز على اتجاهات الاستماع لأشهر 50 فناناً وأكثر 100 أغنية انتشاراً. الهدف هو تحويل البيانات الخام إلى رؤى بيانية (Insights) توضح العوامل المشتركة لنجاح المحتوى الموسيقي.

## 🛠️ Tech Stack (الأدوات المستخدمة)
I utilized a powerful combination of data analysis tools to ensure accuracy and visual appeal:
* **Microsoft Excel:** Used for advanced data cleaning, creating **Pivot Tables**, and designing interactive **Dashboards**.
* **SQL Server:** Applied for querying the Top 100 tracks and filtering artist performance metrics.
* **Python (Pandas):** Employed for data manipulation and exploring statistical correlations between tracks.

## 🚀 Key Features & Workflow
1.  **Data Cleaning:** Processing raw CSV files (Top 100, 50 Artists) to remove duplicates and handle missing values.
2.  **Exploratory Data Analysis (EDA):** Identifying the most streamed genres and artists using SQL queries.
3.  **Visualization:** Building a professional dashboard in Excel to visualize the "Top 50 Artists" performance.
4.  **Statistical Insights:** Analysis of the relationship between track features and their ranking in the Top 100.

## 📂 Project Structure (محتويات المشروع)
* `Data/`: Contains the cleaned CSV files (`top_100.csv`, `50_Artists.csv`).
* `Excel/`: Includes the original `Final project.xlsx` with Dashboards and Pivot Tables.
* `SQL/`: SQL scripts used for data extraction and sorting.
* `Python/`: Jupyter Notebooks for data processing.

## 📊 Key Insights
* Identifying the peak performance periods for the top 50 artists.
* Distribution of track rankings based on streaming volume.
